# Expense Tracker - Naan Mudhalvan Project
**May 2026**

**Developed by:** Yeswathi CM

### About Project
A full-stack web application to track daily expenses with complete CRUD functionality. Built using Python Flask and SQLite for Naan Mudhalvan program.

### Tech Stack
- **Backend:** Python, Flask
- **Database:** SQLite3
- **Frontend:** HTML5, CSS, Jinja2 Template Engine
- **Tools:** VS Code, Git, GitHub

### Key Features
1. Add new expenses with date, category, amount, description
2. View all expenses in dynamic table format
3. Update existing expense details
4. Delete expenses 
5. Real-time total expense calculation
6. Secure database operations using parameterized SQL queries

### How to Run Locally
1. Clone the repo: `git clone https://github.com/yeswathimuthukumaran-oss/the-syntax-creators.git`
2. Install Flask: `pip install flask`
3. Run the app: `python app.py`
4. Open browser: `http://127.0.0.1:5000`
